Exploring the Influence of Higher Education on Sustainable Entrepreneurial Intention in Latin American Countries

This repository contains the replication materials for the study:

Exploring the Influence of Higher Education on Sustainable Entrepreneurial Intention in Latin American Countries: A Mixed-Method Approach

The study examines how higher education influences Sustainable Entrepreneurial Intention (SEI) among university students in Chile, Colombia, and Peru. In particular, it evaluates the role of curricular training, institutional context, subjective norms, attitude toward sustainable entrepreneurship, and perceived behavioral control.

Repository Contents

* SEI_data.txt
    Anonymized quantitative dataset used for the PLS-SEM analysis.
* SEI_data.txt.ok
    SmartPLS metadata file associated with the dataset.
* SEI_data.txt.settings
    SmartPLS dataset configuration file.
* SEI.splsm
    Structural equation model estimated using SmartPLS 4.
* Instrument.pdf
    Survey instrument used to collect the quantitative data.
* data_dictionary.md
    Description of the constructs, variables, indicators, and coding used in the dataset.
* convert_to_csv.py
    Python script to convert the original dataset into CSV format.

Study Overview

* Research design: Explanatory sequential mixed-method design
* Quantitative approach: Cross-sectional survey and Partial Least Squares Structural Equation Modeling (PLS-SEM)
* Sample: 806 undergraduate students
* Countries: Chile, Colombia, and Peru
* Institutions: Universidad Católica del Norte, Universidad del Rosario, and Universidad ESAN
* Data collection period: April–November 2024
* Software: SmartPLS 4

Main Constructs

The dataset includes item-level indicators for the following constructs:

* Attitude Toward Sustainable Entrepreneurship (ATSE)
* Perceived Behavioral Control (PBC)
* Subjective Norms (SN)
* Entrepreneurship Education: Curricular Training (EECT)
* Entrepreneurship Education: Institutional Context (EEIC)
* Sustainable Entrepreneurial Intention (SEI)

Reproducibility

To reproduce the quantitative analysis:

1. Download or clone this repository.
2. Open SmartPLS 4.
3. Import SEI_data.txt as the dataset.
4. Open the structural model file SEI.splsm.
5. Run the PLS Algorithm.
6. Run bootstrapping using 5,000 subsamples and a two-tailed test.
7. Evaluate the measurement model, structural paths, indirect effects, explanatory power, and multigroup comparisons.

Data Availability and Ethical Considerations

The quantitative dataset has been anonymized and does not contain direct participant identifiers.

The data are shared exclusively for academic replication, verification, and non-commercial research purposes. Users should appropriately cite the associated article and acknowledge the original authors when using these materials.

Qualitative transcripts, interview recordings, focus-group materials, and other potentially identifiable contextual information are not publicly available because of confidentiality obligations and ethical restrictions.

The quantitative phase of the study was approved by the Ethics Committee of Universidad Católica del Norte under Resolution No. 09/2024.
